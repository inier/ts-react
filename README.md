# ts-react
typescript + react的简单起手模板

### 关键词
- typescript
- react
- react-router-4
- hot-reload
- mobx
- async component
- svg icon
- 接入antd并可按需加载

### 建议安装
- [mobx chrome devtool](https://chrome.google.com/webstore/detail/mobx-developer-tools/pfgnfdagidkfgccljigdamigbcnndkod) (非必须)

### 运行
1. npm i
2. npm run dev
